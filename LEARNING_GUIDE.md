# Learning Guide

This guide provides a structured approach to working through the exercises.

## Week-by-Week Plan

### Week 1: JavaScript Fundamentals (Part 1)
**Days 1-2: Variables and Types**
- Complete `01-variables-and-types.js`
- Practice type checking and conversion
- Experiment with different data types

**Days 3-4: Functions**
- Complete `02-functions.js`
- Master arrow functions
- Understand closures

**Days 5-7: Arrays**
- Complete `03-arrays.js`
- Practice all array methods
- Solve additional array problems

### Week 2: JavaScript Fundamentals (Part 2)
**Days 1-2: Objects**
- Complete `04-objects.js`
- Practice destructuring
- Work with Object methods

**Days 3-5: Async Programming**
- Complete `05-async-promises.js`
- Understand promises thoroughly
- Practice async/await patterns

**Days 6-7: ES6+ Features**
- Complete `06-es6-features.js`
- Learn modern JavaScript syntax
- Practice with classes

### Week 3: React Fundamentals (Part 1)
**Days 1-2: Components**
- Complete `01-components.jsx`
- Understand JSX syntax
- Practice component composition

**Days 3-4: State and Hooks**
- Complete `02-state-and-hooks.jsx`
- Master useState
- Build interactive components

**Days 5-7: Effects**
- Complete `03-effects.jsx`
- Understand useEffect
- Practice cleanup patterns

### Week 4: React Fundamentals (Part 2)
**Days 1-2: Forms**
- Complete `04-forms.jsx`
- Master controlled components
- Implement form validation

**Days 3-5: Advanced Hooks**
- Complete `05-advanced-hooks.jsx`

- Optimize with useMemo, useCallback

**Days 6-7: Custom Hooks**
- Complete `06-custom-hooks.jsx`
- Create reusable hooks
- Build a mini-project

## Daily Routine

1. **Read (15 minutes)**
   - Review the exercise file
   - Read the topic documentation

2. **Code (45 minutes)**
   - Implement the TODO items
   - Run tests frequently
   - Experiment with variations

3. **Test (15 minutes)**
   - Run all tests
   - Debug any failures
   - Verify edge cases

4. **Review (15 minutes)**
   - Compare with solutions
   - Note areas for improvement
   - Update your learning journal

## Learning Tips

### For JavaScript
- **Console is your friend**: Use `console.log()` liberally
- **Read error messages**: They usually tell you exactly what's wrong
- **Use MDN**: Mozilla Developer Network has excellent docs
- **Practice regularly**: Even 30 minutes daily is better than marathon sessions

### For React
- **Think in components**: Break UIs into small, reusable pieces
- **Understand data flow**: Props down, events up
- **Master hooks**: They're the foundation of modern React
- **Read the docs**: React's official docs are excellent

## Getting Help

If you get stuck:
1. Read the error message carefully
2. Check the solution file for hints
3. Search the error on Google/Stack Overflow
4. Review the relevant documentation
5. Take a break and come back fresh

## Measuring Progress

You're making good progress when you can:
- [ ] Explain concepts in your own words
- [ ] Solve exercises without looking at solutions
- [ ] Debug errors independently
- [ ] Build small projects from scratch
- [ ] Help others understand concepts
