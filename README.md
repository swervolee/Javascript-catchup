## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/swervolee/Javascript-catchup.git
cd Javascript-catchup
```

2. Install dependencies:
```bash
npm install
```

### Running Tests

Run all tests:
```bash
npm test
```

Run tests in watch mode:
```bash
npm run test:watch
```

Run tests with coverage:
```bash
npm run test:coverage
```

## ğŸ“– Learning Path

### JavaScript Fundamentals

1. **Variables and Types** - Learn about `let`, `const`, `var`, and JavaScript types
2. **Functions** - Master function declarations, arrow functions, and higher-order functions
3. **Arrays** - Practice array methods like `map`, `filter`, `reduce`, and more
4. **Objects** - Work with object creation, methods, and destructuring
5. **Async/Promises** - Understand promises, async/await, and error handling
6. **ES6+ Features** - Explore modern JavaScript features

### React Fundamentals

1. **Components** - Create functional components and understand JSX
2. **State and Hooks** - Use `useState` for managing component state
3. **Effects** - Master `useEffect` for side effects and lifecycle
4. **Forms** - Handle form inputs and validation
5. **Advanced Hooks** - Learn `useContext`, `useReducer`, `useMemo`, `useCallback`, `useRef`
6. **Custom Hooks** - Create reusable custom hooks
